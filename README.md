This is a simple 2-player Tic Tac Toe game built in Python. The game runs in the command line where two players take turns to mark their positions on a 3x3 grid.

The first player to align three of their symbols (X or O) horizontally, vertically, or diagonally wins the game. If all positions are filled and no player wins, the game ends in a draw.

This project is great for beginners to practice logic building and core programming concepts.

🚀 Features

2-player gameplay (X vs O)

Command-line interface (CLI)

Input validation

Win and draw detection

Simple and clean code

🛠️ Requirements

Python 3.x

▶️ How to Run

Clone the repository:

git clone https://github.com/your-username/tic-tac-toe-python.git

Go to the project folder:

cd tic-tac-toe-python

Run the game:

python game.py

🎯 How to Play

The game is played between two players: X and O

Players take turns choosing a position from 1 to 9

The first player to get 3 marks in a row wins

If all cells are filled and no winner → it's a draw 🤝

🧩 Board Layout

1 | 2 | 3
--+---+--
4 | 5 | 6
--+---+--
7 | 8 | 9
